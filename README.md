# 🍽️ Food Chooser Web App

A simple, beginner-friendly web application that helps you decide what to eat! It randomizes food options based on your favorite cuisine and dietary preferences.

## ✨ Features
- **Cuisine Selection:** Choose from Punjabi, Maharashtrian, South Indian, or White People Food.
- **Dietary Filter:** Toggle between **Veg** and **Non-Veg** (strictly Chicken and Fish only—no beef).
- **Extensive Veggie List:** Includes classic Maharashtrian vegetables like Dodka, Vanga, Gilka, and more.
- **Mobile Friendly:** Works on both desktop and mobile browsers.

## 🚀 How to Use
1. Select your diet (Veg or Chicken/Fish).
2. Pick a cuisine from the dropdown menu.
3. Click the **"What's for dinner?"** button to get a random suggestion!

## 🛠️ Built With
- **HTML5:** For the structure.
- **CSS3:** For the clean, beginner-friendly styling.
- **JavaScript:** For the randomization logic and data handling.

## 📂 Project Structure
- `index.html`: The main code containing the structure, styles, and logic.
